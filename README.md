# Keiba AI

AI競馬フォーメーション予想アプリ。血統・脚質・展開・ペース・騎手・地力（重賞過去実績）などを総合分析し、軸馬・ヒモ馬の買い目を自動生成します。

## 公開URL

https://keiba-ai-zeta.vercel.app/

## 主な機能

| 機能 | 説明 |
|------|------|
| フォーメーション予想 | 軸馬・ヒモ馬の自動選出と買い目（3連複）生成 |
| 展開・ペース分析 | 脚質分布からハイペース/スローペースを推定し展開有利馬を抽出 |
| AI予想まとめ | グレード・距離・ペース・逃げ馬構成・軸信頼度・穴馬をレース別コメントで要約 |
| 地力スコア | 重賞過去実績（着順・出走頭数・グレード）から算出した加点項目 |
| 騎手スコア | jockey_statsの複勝率を騎手評価値として使用 |
| 重賞カレンダー | 2026年重賞一覧。過去レースはJST基準でグレーアウト |
| レース結果比較 | 結果入力後に予想的中判定を表示 |
| 馬マスタ一覧 | 出走馬の脚質・父系・母父系・血統系統を一覧表示 |
| 競馬用語集 | 買い目・脚質・展開・重賞などの用語解説 |

## 予想ロジック概要 (v9.2)

重賞レース専用。人気（オッズ）は一切使用しない回収率重視の設計。

**スコア構成（軸馬）**

```
軸スコア = 血統スコア × 脚質×展開適合 + 騎手スコア + 地力スコア + 頭数補正
```

**スコア構成（ヒモ馬）**

```
ヒモスコア = 血統適性 + 脚質展開 + 騎手 + 地力 + 穴馬ボーナス
```

**地力スコア**（直近重賞実績を加重平均）

- G1: 重み1.5 / G2: 1.2 / G3: 1.0 / OP以下: 0.8
- 着順 / 出走頭数 の相対値をスコア化。レンジは ±0.08

**AI注目の穴馬**（人気外のヒモ候補から選出、軸馬は除外）

## ページ構成

```
/                  トップ（レース一覧・重賞カレンダー）
/race/[id]         レース予想ページ（フォーメーション・AI分析・買い目）
/horses            馬マスタ一覧
/glossary          競馬用語集
/how-to-buy        買い方ガイド
/login             ログイン（Supabase Auth）
/design            デザインシステム確認ページ
```

## DBテーブル構成（主要）

| テーブル / ビュー | 内容 |
|------------------|------|
| `races` | レース情報（名称・日付・グレード・距離・コース等）|
| `horses` | 馬マスタ（脚質・血統系統等）|
| `entries` | 出走情報（馬番・騎手・前走3F・馬体重・着順）|
| `race_results` | 確定着順（AI精度検証用）|
| `jockey_stats` | 騎手別複勝率（place3_rate）|
| `horse_style_profiles` | 馬ごとの脚質プロファイル |
| `horse_form_view` | 重賞過去実績ビュー（地力スコア計算用）|

## セットアップ

```env
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```

```bash
npm run dev   # http://localhost:3000
```
