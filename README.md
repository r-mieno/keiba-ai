# 競馬AI予想（研究プロジェクト）

AIを用いた競馬レース分析アプリのプロトタイプ。

## 目的

AI駆動開発の検証として、競馬レースの分析と買い目生成を行うアプリを個人開発しています。

## 主な機能

- レース展開予想
- AI戦略提案
- 展開有利馬の抽出
- AI注目の穴馬
- 買い目提案
- レース結果との比較

## 技術スタック

- Next.js
- Supabase
- Vercel
- GitHub

## 公開URL

https://keiba-ai-zeta.vercel.app/

## このプロジェクトについて

AI駆動開発の実験として、個人で構想・設計・実装しています。
社内共有用のプロトタイプとして公開しています。
