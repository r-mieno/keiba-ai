# Keiba AI

AI競馬フォーメーション予想アプリ。血統・脚質・展開・ペース・騎手・地力などを総合分析し、軸馬・ヒモ馬の買い目を自動生成します。

## 公開URL

https://keiba-ai-zeta.vercel.app/

## 主な機能

| 機能 | 説明 |
|------|------|
| フォーメーション予想 | 軸馬・ヒモ馬の自動選出と買い目（3連複）生成 |
| 展開・ペース分析 | 脚質分布からハイペース/スローペースを推定し展開有利馬を抽出 |
| AI予想まとめ | グレード・距離・ペース・逃げ馬構成・軸信頼度・穴馬をレース別コメントで要約 |
| 地力スコア | 重賞過去実績（着順・出走頭数・グレード）から算出した加点項目 |
| 騎手スコア | 3着内率 + 重賞勝利ボーナスで算出 |
| 重賞カレンダー | 2026年重賞一覧。過去レースはJST基準でグレーアウト |
| レース結果比較 | 結果入力後に予想的中判定を表示 |
| 馬マスタ一覧 | 出走馬の血統系統・直近走行データ・重賞実績を一覧表示 |
| 競馬用語集 | 買い目・脚質・展開・重賞などの用語解説 |

## 予想ロジック v10（デバッグ版）

> 本番表示は v9.1 を使用。v10 はデバッグパネルに表示。

重賞レース専用。人気（オッズ）は一切使用しない回収率重視の設計。

### 軸馬スコア式

```
axisScore =
  paceFit        × 0.20   // データ駆動型脚質（4角順位から自動計算）
  jockeyScore    × 0.13   // 騎手スコア（3着内率＋重賞勝利ボーナス）
  place3Rate     × 0.20   // 馬の3着内率（出走数補正あり）
  recentForm     × 0.17   // 近走重賞実績スコア（重賞経験数補正あり）
  closingScore   × 0.10   // 上がり3F速度スコア（horse_form_recordsから）
  stability      × 0.10   // レース安定度（ペース安定ほど高い）
  ＋ postAdj            // 枠順×会場補正（加算項目）
  ＋ venueAdj           // 会場×脚質補正（加算項目）
  ＋ bloodlineBonus     // 血統×距離適性ボーナス（加算項目）
  ＋ groundStrength     // 地力スコア（重賞実績から±補正）
  ＋ weightAdj          // 斤量補正（加算項目）
```

### ヒモ馬スコア式

```
himoScore =
  paceFit        × 0.20
  jockeyScore    × 0.13
  place3Rate     × 0.30   // 3着内率（軸より重視）
  recentForm     × 0.17
  closingScore   × 0.15   // 上がり（古馬戦）
  stability      × 0.05
  ＋ 加算項目（軸と同じ）
```

## パラメータ詳細

### `paceFit`（データ駆動型ペース適性）
- `horse_form_records.corner_pos`（4角通過順位）から自動計算
- 手入力の脚質分類は**使わない**
- 4角順位を頭数(16)で割り正規化 → 前傾度(0=逃げ, 1=追い込み)
- ペースと前傾度を掛け合わせてスコア化（0.34〜0.66）
- データなし: 0.50（ニュートラル）
- **ハイペース(fast)** のとき逃げ馬(front)は**1頭制限**

### `jockeyScore`（騎手スコア）
- ベース: `jockey_stats.place3_rate`（2026年度3着内率）
- ボーナス加算: G1勝利×0.02（上限+0.06）、G2×0.01（+0.03）、G3×0.005（+0.02）
- 未登録騎手: デフォルト 0.25

### `place3Rate`（馬の3着内率）
- `horses.place3_rate`（2026年度の3着内率を手入力）
- **出走数補正**: `race_count`が少ないほど全体平均(0.33)に引き寄せる
  - 信頼度 = min(1, race_count / 15)
  - 例: 4戦 → 信頼度0.27、p3rate 1.0 → 補正後 0.51

### `recentForm`（近走重賞実績）
- `horse_past_results`（重賞成績テーブル）から計算
- 計算式: `0.5 + GRADE_W × finishBonus`（ベース0.5の加点方式）
- データなし・条件戦のみ: 0.5（ニュートラル）
- **重賞経験数補正**: 1戦=33%信頼度、2戦=67%、3戦以上=100%

| グレード | GRADE_W |
|---------|---------|
| G1 | 1.00 |
| G2 | 0.65 |
| G3 | 0.45 |

| 着順 | finishBonus | G1スコア | G2スコア | G3スコア |
|------|------------|---------|---------|---------|
| 1着 | 0.50 | 1.000 | 0.825 | 0.725 |
| 2着 | 0.35 | 0.850 | 0.728 | 0.658 |
| 3着 | 0.25 | 0.750 | 0.663 | 0.613 |
| 4〜5着 | 0.10 | 0.600 | 0.565 | 0.545 |
| 6着以下 | 0.02 | 0.520 | 0.513 | 0.509 |

### `closingScore`（上がり3F速度）
- `horse_form_records.last3f`（直近最大5走）から計算
- 33.0秒→1.0（最速）、36.5秒→0.0（最遅）で正規化
- 0.35〜0.65に圧縮（突出馬の過大評価を防ぐ）
- データなし: 0.50（非圧縮ニュートラル）

### `stability`（レース安定度）
- 出走馬の脚質分布からペースの安定度を計算
- 逃げ馬が多い・ペースが混乱するほど低い
- 値: 0〜100 → `1 - stability/100` でスコア化

### `postAdj`（枠順×会場補正）
- 馬番と会場から内枠/外枠の有利不利を加算

| 会場 | 内枠(1-4) | 中内(5-9) | 中外(10-14) | 外枠(15+) |
|------|---------|---------|-----------|---------|
| 阪神マイル | +0.020 | +0.010 | -0.010 | -0.020 |
| 阪神中距離 | +0.015 | +0.005 | -0.005 | -0.015 |
| 中山 | +0.030 | +0.010 | -0.015 | -0.025 |
| 東京マイル | +0.010 | +0.005 | -0.005 | -0.010 |
| 東京中長距離 | +0.005 | 0 | 0 | -0.005 |
| 京都・中京 | +0.010 | +0.005 | -0.005 | -0.010 |

### `groundStrength`（地力スコア）
- `horse_past_results` の重賞実績から±補正
- レース距離との近さも考慮
- 範囲: 約 -0.08 〜 +0.08

### `bloodlineBonus`（血統×距離適性）
- `horses.father_line` + `damsire_line` × 距離バケット
- 父系・母父系の組み合わせでクロスボーナスあり
- 範囲: 0 〜 約 +0.05

### `weightAdj`（斤量補正）
- 標準斤量(55kg)からの差分で±補正
- 範囲: 約 -0.02 〜 +0.02

## データ入力の種類と用途

| データ | テーブル | 入力場所 | 用途 |
|--------|---------|---------|------|
| 3着内率 | `horses.place3_rate` | 馬マスタ | place3Rate |
| 出走数 | `horses.race_count` | 馬マスタ | place3Rate信頼度補正 |
| 父系・母父系 | `horses.father_line` | 馬マスタ | bloodlineBonus |
| 重賞実績 | `horse_past_results` | 馬マスタ | recentForm, groundStrength |
| 直近走行データ | `horse_form_records` | 馬マスタ | paceFit, closingScore, 自動脚質判定 |
| 騎手3着内率 | `jockey_stats.place3_rate` | 騎手マスタ | jockeyScore |
| 騎手重賞勝利数 | `jockey_stats.g1/g2/g3_wins` | 騎手マスタ | jockeyScore bonus |
| 馬番・騎手名 | `entries` | レース管理 | postAdj, jockeyScore |

## DBテーブル構成（主要）

| テーブル | 内容 |
|---------|------|
| `races` | レース情報（名称・日付・グレード・距離・会場等）|
| `horses` | 馬マスタ（血統系統・3着内率・出走数）|
| `entries` | 出走情報（馬番・騎手・上がり3F・馬体重・着順）|
| `race_results` | 確定着順（AI精度検証用）|
| `jockey_stats` | 騎手別3着内率・重賞勝利数 |
| `horse_past_results` | 重賞過去実績（grade・距離・着順・頭数）|
| `horse_form_records` | 直近走行データ（上がり3F・4角順位・着順）|
| `horse_style_profiles` | 手動脚質設定（v10では未使用）|
